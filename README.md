# Portfolio Site

My personal portfolio website built with Node.js, Express, and Bootstrap.

## Links

Deployment: https://portfolio-site-2025.onrender.com/
Repository: https://github.com/joshonline/portfolio-site-2025

## What's Inside

- **Home**: Quick intro and featured projects
- **About**: A bit about me, my skills, and what I'm into
- **Projects**: Stuff I've built (networking, IT, and code)
- **Contact**: Ways to reach me

## Tech Stack

- Node.js + Express
- EJS templating
- Bootstrap 5
- Font Awesome icons

## Running Locally

```bash
npm install
npm start
```

Then visit `http://localhost:3000`

## Project Structure

- `/routes` - Page routes
- `/views` - EJS templates
- `/public` - Static files (CSS, images, etc.)

## Git Workflow

Using GitHub Flow:

- `main` - Production branch
- `dev` - Development branch
- `feature/*` - New features
- `improvement/*` - Refactors and improvements
